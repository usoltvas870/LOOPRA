# Scoring Logic — Nura TikTok Viral Screening Radar

## Метрики

### 1. Engagement Rate (ER)

```
ER = (likes + comments + shares) / views
```

Доля зрителей, которые совершили действие. **Хороший ER:**

| ER | Оценка |
|---|---|
| > 25% | Отлично — каждый 4-й зритель лайкнул/репостнул |
| 15–25% | Хорошо — средний уровень для ниши |
| 5–15% | Нормально — контент смотрят, но не вовлекаются |
| < 5% | Слабо — низкая вовлечённость |

Пример: 203K views, 64.7K likes + 96 comments + 3.8K shares → ER = 33.78%

### 2. Comment Density (CD)

```
CD = comments / views
```

Показывает, насколько контент провоцирует обсуждение. **Хороший CD:**

| CD | Оценка |
|---|---|
| > 1% | Отлично — много споров/обсуждений |
| 0.3–1% | Хорошо |
| < 0.3% | Слабо — мало комментариев |

Пример: 2.2M views, 1.6K comments → CD = 0.07% (слабо для таких охватов)

### 3. Viral Score

```
viral_score = views / author_followers
```

Показывает, во сколько раз просмотры превышают подписчиков. Если у автора 1000 подписчиков, а видео набрало 100K просмотров — viral_score = 100. **Это главный индикатор "залета".**

| viral_score | Значение |
|---|---|
| > 100 | Вирусный взрыв — видео ушло далеко за пределы аудитории |
| 10–100 | Хороший разлёт |
| 1–10 | В рамках подписчиков |
| < 1 | Ниже среднего по аудитории |

**Важно:** метрика доступна, только если у автора открыто число подписчиков. Если нет — не учитывается.

### 4. Normalized Views

```
views_norm = views / max_views_в_выборке
```

Приводит просмотры к диапазону 0–1 относительно лучшего видео в текущем запуске.

### 5. Subscriber Potential (0–10)

```
subscriber_potential = views_norm × 6 + er_norm × 3 + cd_norm × 1
```

Где:
- `views_norm = views / max_views_в_выборке` (нормированный охват)
- `er_norm = min(ER / 0.3, 1.0)` (ER 30% и выше — идеал, даёт полные 3 балла)
- `cd_norm = min(CD / 0.01, 1.0)` (1% comment density — идеал, даёт 1 балл)

**Зачем:** Final Score ищет качественный виральный контент. Subscriber Potential ищет **ролики, которые приведут подписчиков**. Это метрика для посева и рекламы.

| Subscriber Potential | Оценка |
|---|---|
| > 7 | Хит — максимум охвата и вовлечения |
| 4–7 | Хороший кандидат для продвижения |
| 2–4 | Средний — можно использовать в ленте |
| < 2 | Низкий охват — не для посева |

**Пример из отчёта:**

Ролик #11 (4.4M views, ER 18.97%):
```
views_norm = 4400000 / 4400000 = 1.0
er_norm = min(0.1897 / 0.3, 1.0) = 0.632
cd_norm = min(0.00097 / 0.01, 1.0) = 0.097

Subscriber Potential = 1.0 × 6 + 0.632 × 3 + 0.097 × 1 = 6 + 1.896 + 0.097 = 7.99
```

Ролик #1 (203K views, ER 33.78%):
```
views_norm = 203100 / 4400000 = 0.046
er_norm = min(0.3378 / 0.3, 1.0) = 1.0

Subscriber Potential = 0.046 × 6 + 1.0 × 3 = 0.276 + 3 = 3.28
```

**Итог:** #11 приведёт в 2.4 раза больше подписчиков, чем #1, несмотря на то что у #1 выше Final Score.

---

## Когда какую метрику использовать

| Задача | Какая метрика важна |
|---|---|
| Выбрать формат для контента под Nura | Final Score |
| Выбрать видео для таргета / посева | Subscriber Potential |
| Оценить качество контента | Engagement Rate |
| Найти видео, улетевшее в рекомендации | Viral Score |
| Понять, о чем спорят в комментариях | Comment Density |

## Ограничения формулы

```
Final Score = views_norm × 0.5
            + ER × 100 × 0.3
            + CD × 1000 × 0.2
            + viral_bonus (опционально, +0.1)
```

### Разбор весов

| Компонент | Вес | Зачем |
|---|---|---|
| `views_norm × 0.5` | 50% | Охват — главный сигнал вирусности |
| `ER × 100 × 0.3` | 30% | Вовлечение — качество контента |
| `CD × 1000 × 0.2` | 20% | Обсуждение — глубина интереса |
| viral_bonus | +0.1 | Бонус, если видео ушло далеко за подписчиков |

Множители 100 и 1000 нужны, чтобы привести ER (0.00–1.00) и CD (0.000–0.01) к сопоставимой с views_norm шкале.

### Пример расчёта

Ролик #1 из отчёта:
- Views: 203,100 | max_views в запуске: 4,400,000
- ER: 33.78% (0.3378)
- CD: 0.047% (0.00047)

```
views_norm = 203100 / 4400000 = 0.0462
ER_term    = 0.3378 × 100 × 0.3 = 10.134
CD_term    = 0.00047 × 1000 × 0.2 = 0.094

Final Score = 0.0462 × 0.5 + 10.134 + 0.094
           = 0.0231 + 10.134 + 0.094
           = 10.3302
```

Ролик выиграл не за счёт охвата (всего 203K против 4.4M лидера), а за счёт **экстремально высокой вовлечённости** — 33.78% ER.

## Какие Final Score считать хорошими

По данным первого запуска (30 роликов, views ≥ 10K):

| Score | Оценка | Примеры |
|---|---|---|
| > 10 | Вирусный хит | #1 — 10.33 (33% ER, 203K views) |
| 6–10 | Сильный кандидат | #3 — 7.66 (24% ER, 595K views) |
| 5–6 | Выше среднего | #11 — 6.01 (18% ER, 4.4M views) |
| 4–5 | Средний | #24 — 4.89 (15% ER, 1.2M views) |
| < 4 | Ниже среднего | — |

**Что важнее охвата:**
Ролик #11 (4.4M views, Score 6.01) проиграл ролику #1 (203K views, Score 10.33), потому что ER в 2 раза ниже. **Формула штрафует контент, который много смотрят, но мало вовлекаются.**

## Ограничения формулы

1. **Нет нормировки по нише.** ER 5% для матрицы судьбы и ER 5% для dark feminine — не одно и то же. Со временем можно добавить бенчмарки по хэштегам.
2. **Не учитывает длительность.** Короткие ролики имеют более высокий ER по определению.
3. **Не учитывает время.** Ролик, собравший 1M за сутки, ценнее ролика с 1M за месяц.
4. **Viral Score — опционально.** TikTok прячет подписчиков у многих аккаунтов.
5. **Multiplier-эффект.** Множители 100 и 1000 подобраны эмпирически для первого запуска. Могут меняться под твою нишу.
